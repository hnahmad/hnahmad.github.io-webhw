Explanation of the Project

This is a homework assignment for Data Analytics and Visualization bootcamp. It is referencing a climate assignment we did previously. We are creating a landing page, 4 visualization pages, and more. The assignment we are referencing is an assignment analyzing how weather changes at various latitudes (and as you get closer to the equator). We assembled a dataset of more than 500 cities and looked at four factors:

  1. Temperature
  2. Humidity
  3. Cloudiness
  4. Wind Speed
